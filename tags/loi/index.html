Nos Deputes

Novembre 4, 2018 dans AN
Visualiser le parlement ggparliament est une extension de ggplot2 pour visualiser des donnees parliamentaires. Ce paquet R permets de visualiser different types d’arrangement de salle de legislature comme l’hemycycle ( à la francaise), le Fer à chevel (en australie), les Banc en opposition (en Angleterre), le cercle (en Russie) ou la salle de classe Dans un parlement, une salle des séances disposée en hémicycle doit, théoriquement, être destinée à favoriser les consensus entre les groupes politiques surtout lorsqu’il s’agit d’un système démocratique où le multipartisme domine.
